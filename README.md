# CSS Grid Fashion Store UI 👗✨

Salam! Bu repository mənim müasir veb-dizayn arxitekturasının ən güclü alətlərindən biri olan **CSS Grid Layout** texnologiyasını dərindən öyrənmək və tətbiq etmək üçün yığdığım "MixFashion" adlı moda və geyim e-ticarət şablonudur. Layihədə həm mürəkkəb Grid strukturlarından, həm də **Bootstrap 5**-in təqdim etdiyi responsiv xüsusiyyətlərdən istifadə etmişəm.

## 🛠 Nələri öyrəndim və tətbiq etdim?

- **CSS Grid Arxitekturası:** Ənənəvi Flexbox və ya Float üsullarından fərqli olaraq, səhifədəki mürəkkəb foto qalereyaları, məhsul bannerlərini və asimmetrik düzülüşləri (məsələn, böyük və kiçik şəkillərin bir arada olması) təmiz bir şəkildə qurmaq üçün CSS Grid-dən istifadə etdim.
- **Moda E-ticarət Dizaynı:** Geyim mağazalarına xas olan "Baxış Kitabçası" (Lookbook) tərzi böyük bannerlər, karusellər (slide-show) və estetik məhsul nümayişi blokları hazırladım.
- **Bootstrap 5 İnteqrasiyası:** Grid ilə yanaşı, layihənin sürətli naviqasiya, düymələr və baza mobil uyğunluq (Responsive) ehtiyaclarını qarşılamaq üçün lokal Bootstrap 5 fayllarını sistemə daxil etdim.
- **Mürəkkəb Qalereya Quruluşu:** Müxtəlif ölçülü şəkilləri (`img6_...webp`, `13_...webp` və s.) bir-birinin üzərinə çıxmadan, ekrana tam oturan, xüsusi aralıqlarla (gap) nizamlanmış bir şəbəkədə (grid) yerləşdirdim.
- **Müasir Şəkil Formatları:** Veb-saytın daha sürətli yüklənməsi üçün standart JPEG əvəzinə, müasir və yüngül `.webp` şəkil formatlarından aktiv istifadə etdim.
